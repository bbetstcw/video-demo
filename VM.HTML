  • 与本地环境实现一致的混合环境

  • 支持 Microsoft SQL Server 和 SharePoint Server

  • 按分钟计费，节约成本

SQL Server

使用 SQL Server 团队构建的映像，您可以在几分钟内轻松设置 SQL Server。您可以使用针对快速开发/测试环境提供的免费 MSDN 许可证创建虚拟机，也可以使用 SQL Server AlwaysOn 跨多个 Windows Azure 区域部署复杂的生产应用程序。

开放，提供更多选择

您可以在 Windows Azure 上部署全套开放的以及社区驱动的操作系统以及软件解决方案。借助 Windows Azure，您可以选择 Ubuntu 和 SUSE 等各类 Linux 发行版， Windows Azure 是开放的，为您提供了很多选择。

省钱，获得更高灵活性

针对快速计算型工作负载、快速开发和测试环境，以及频繁停止和启动等场景，在 Windows Azure 中部署虚拟机时，您将按分钟付费。包括针对 Windows、Linux和SQL Server的计费皆是按分钟收取。您甚至可以让整个平台根据您的计划或实例的 CPU 使用率自动缩放。我们将提供能满足您需要的基础结构和软件计费模型，而不是为了满足我们的需要！

混合连接

使用虚拟网络，您可以控制和配置您网络的方方面面，定义子网和首选 DNS IP。您可以通过互联网，用安全的方式通过 VPN 在 Windows Azure 中连接您的虚拟机。

此服务包含跨多个实例的负载平衡，且易于配置。您可以创建负载平衡器，添加安全 ACL 以便控制访问权限，并定义具体的探测，以便帮您监视应用程序运行状况，所有这些都可以在易于使用的门户中直接进行。

更快、更大且更好的 D 系列虚拟机

您现在可以部署更大的 D 系列虚拟机，满足您的工作负载的需要。这种大小的虚拟机运行速度更快，各方面的指标更强大，处理器速度更快、内存更大，提供基于 SSD 的存储所支持的临时磁盘，方便虚拟机进行极快速的临时存储。快来试一试吧！今天就可以体验。

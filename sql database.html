构建支持大规模的 SaaS 应用程序

构建软件即服务 (SaaS) 应用程序的开发人员可利用 SQL 数据库提供灵活性以支持爆炸性增长和盈利的商业模型。对于具有不可预测的数据库资源消耗的工作负载，弹性数据库模型可使您在一组数据库之间均分要利用的资源，无需超量配置数据库以满足峰值需求，反之，您可以使用弹性数据库池来让成百或成千的数据库利用您控制预算内的资源。借助可跨数据库组控制价格和性能的购买模型，您可以推动成本效益。

轻松管理大量数据库

当您扩展到数以千计的数据库时，简单的管理任务也会变得复杂，这是爆炸性增长带来的挑战。随着业务不断增长，SaaS 开发人员将花费大量时间编写复杂逻辑以应对这些架构变化以及其他管理运营问题。使用弹性数据库，生活变得更轻松。您选取一个脚本，将它作为作业提交，Azure SQL 数据库将完成跨多个数据库执行脚本的剩余工作。弹性数据库池可以简化跨许多数据库缩放的应用程序的生成和管理，因此，生成针对单个数据库或上千数据库的应用程序就和使用熟悉的 T-SQL 和 ADO.NET 编程模型一样轻松。您还可以运行集中式查询操作，例如跨多个数据库的报告和数据提取、返回单个统一结果集。

支持弹性扩展，提供更可预测的性能

在 Windows Azure 强大功能的支持下，高吞吐量应用程序可以在上千个数据库之间灵活地扩展。新增的 SQL 数据库服务级别使应用程序能够轻松地在每个数据库上扩展或收缩，获得可预测的性能。Windows Azure SQL 数据库的弹性扩展预览版可简化应用程序的开发与管理，并可在大量数据库之间横向扩展。

Flavorous 使用支持大规模扩展的混合架构连接内部与云端环境，应对暂时的峰值负载，开拓新业务。

加强关键应用程序的业务连续性

通过内建的副功能，以及数据库层面所提供的 99.99%* 服务级别协议（由 世纪互联 支持），对数据可用性更放心。使用不同服务级别内建的连续性选项，可获得更好的应用程序连续性，面对灾难性事件获得妥善保护。SQL 数据库高级版提供了主动地域复制功能，可以让你在任何 Windows Azure 区域创建多达 4 个可读辅助副本，并控制在何时何处进行故障转移。通过自助式还原可更好地对“不利条件下的恢复”加以控制，客户可以从可用的数据备份副本控制数据还原的全过程。在高级版中，SQL 数据库最多可将备份副本保存 35 天，供您随时进行恢复。

* 99.99% 运行时间服务级别协议仅适用于基本、标准和高级服务级别。SQL 数据库 Web 版和企业版提供 99.9% 的运行时间服务级别协议。

通过部署灾难恢复解决方案和地域冗余，MYOB 可以根据需要扩展到数千个数据库。

服务可自行管理，几乎无需人工维护

SQL 数据库服务中包含自动软件修补功能，你无需进行任何基础结构维护。同时，内置的系统副本功能可帮你提供固有的数据保护，有保障的数据库持续运行时间和系统稳定性，让开发人员和架构师无忧使用。旧计算机发生故障时，系统副本会自动转移到系统配置的新计算机。

Pottermore 将自己的应用程序迁移到 Windows Azure 平台即服务，省略了管理和维护虚拟机的繁重工作。

支持熟悉的工具和平台，提高工作效率

通过可编程 API 驱动开发运营任务（如向外扩展或业务连续性），在扩展到成百上千个数据库的同时简化管理工作。SQL 数据库提供了很多管理工具，如 REST API、PowerShell、支持 HTML5 的 Windows Azure 管理门户，以及 SQL Server Management Studio，此外还支持常见的平台与技术（包括 .NET、Java、PHP、Ruby on Rails 和 Node.js）。通过与 Visual Studio 集成，可以进行无缝在线/离线开发，并在本地应用程序和针对云设计的应用程序上进行开发。通过支持 DAC 框架（允许在本地和云之间迁移），可将现有应用程序扩展到云。

Samsung 针对全球性 SmartTV 业务简化开发工作，成本降低 80%。SQL 数据库高级版支持数千用户同时上网注册市面上领先的 SmartTV 服务。

支持合规相关任务

SQL Database 已通过主要云审核机构的验证，该验证是已包含 Windows Azure ，合规认证和 HIPAA BAA、ISO/IEC 27001:2005、FedRAMP 及欧盟标准条款等审批认证中。

搭配使用不同级别服务，支持创新式设计

SQL 数据库提供了多种不同级别的服务，可支持不同类型的数据库工作负载（从轻型到重型），你可以根据应用程序设计需要更改自己所用的服务级别，或搭配使用不同级别。借助 Windows Azure 的强大功能，你可以根据独特、现代的应用程序设计需求将 Windows Azure 服务与 SQL 数据库搭配使用，以降低成本、提高资源效率，开拓新的业务机遇。
